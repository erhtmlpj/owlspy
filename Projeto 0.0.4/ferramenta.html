Sobre a ferramenta

Funcionalidade

O programa registra os dados do usuário como tela e teclas digitadas e envia direto para a sua conta neste site, o programa também realiza streams do desktop para você

Como utilizar

Instalação

Para instalar, basta entrar na seção de downloads ao lado e procurar a versão que se adequa aos requisitos de seu computador;

Downloads

Utilização

Após ter o programa instalado na pasta que se deseja monitorar, é necessário possuir uma conta neste site e saber o ip privado do usuário. na sua janela de administração, basta inserir o ip do usuário, ou se ele já estiver lá, basta começar o monitoramento,
